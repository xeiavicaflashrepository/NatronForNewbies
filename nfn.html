Introduction To Natron

Natron is a free, libre (Open Source) GPLv2 licensed compositing and visual effects program stylized after nuke. It works on Microsoft Windows, Apple MacOS, and GNU/Linux.

Natron In Action.


There are 3 things to note about the above natron interface:

  1. To the far left, are action buttons to add a node from a group.
  2. To the upper right of that, is a preview window of your project.
  3. To the bottom right of that, is the work area for nodes you're using right now.
  4. To the right of both of them, is configuration of the placed and connected nodes.

What Is A Node?

A node like this:

can be described as like a LEGO brick. You place a piece somewhere and connect them together. This contrasts very heavily from After Effects, which is layer based.
In my personal opinion, despite the steeper learning curve, nodes are far more flexible than layers. To make it less overwhelming, you can group the nodes together.
